🧇 Waffle Fiesta – UPI Token Generator(https://waffle-fiesta.netlify.app/)

A lightweight web application that generates unique tokens (coupons) for customers after completing a UPI payment.
Built specifically for college food stalls and events to simplify order handling without using paid payment gateways.

🚀 **Features**
🔢 Generate unique token for each customer
💳 UPI-based payment workflow (PhonePe / Google Pay / Paytm)
🧾 Token generation after payment confirmation
⚡ Fast and simple UI for quick transactions
📱 Mobile-friendly design for real-time usage



🛠️ **Tech Stack**
Frontend: React.js
Styling: Tailwind CSS
Backend (optional / planned): Supabase
Hosting: Netlify



⚙️ **How It Works**
Customer scans the UPI QR code and makes payment
Customer enters the Transaction ID
System verifies the transaction ID (basic / planned validation)
A unique token (coupon) is generated
Token is shown/downloaded and used for order pickup
